# План 12f — Фильтры качества данных: тренд без выбросов + детект аномалий подписчиков

## Зачем (риск №5 из ПЛАН.md: «не учиться на мусоре»)
Две дыры качества данных, обе подтверждены по коду:
1. Тренд неделя-к-неделе (12a) считается ПО ВСЕМУ окну с выбросами — один рекламный/
   виральный пост даёт ложный «рост охвата» в отчёте и экране «Рост».
2. Ряд подписчиков Telemetr (12e-2) уже в кэше, но резкие скачки (накрутка /
   рекламные всплески) никак не подсвечиваются владельцу.

## Скоуп 12f (один шаг, обе правки маленькие)

### Часть 1 — тренд без выбросов (фикс 12a)
- `src/core/analytics/insights.ts`: сейчас (строки ~43, ~68) `flagViewOutliers`
  считается ТОЛЬКО для текущего окна, а `compareTrend(current, previous)` получает
  оба окна целиком. Фикс: посчитать флаги и для `previous`, отфильтровать выбросы
  из ОБОИХ окон перед `compareTrend`. Список `outliers` в `Insights` (показ владельцу)
  не менять — он и должен показывать выбросы.
- Порог/логика `flagViewOutliers` не трогаем (≥3× медианы, `MIN_SAMPLE_FOR_OUTLIERS=4`).
- Нюанс: окно меньше 4 постов → флагов нет → тренд как раньше (честно, базы мало).

### Часть 2 — детект аномалий ряда подписчиков (новое ядро)
- `src/core/analytics/` или `src/core/market/` → `subscriberAnomaly.ts` (ЧИСТОЕ):
  `detectSubscriberAnomalies(points)` по ряду `SubscriberPoint[]` (уже в кэше 12e-2):
  суточная |Δ| ≥ max(АБС_МИН, ДОЛЯ × база) → аномалия {date, delta}. Пороги —
  константами (предложение: ≥5% базы И ≥10 человек; уточнить на живом ряду
  @sofia_gada1ka — там органика ±1-2/день). Дырки в ряду (Δ за >1 день) — НЕ считать
  суточным скачком (поделить на разрыв дней или пропустить).
- Вывод: строка-предупреждение в секции «🌍 Рынок» (`marketSection.ts`), только если
  аномалии есть: `⚠️ Резкие скачки подписчиков: N за 28д (даты) — возможна накрутка
  или рекламный всплеск`. Автоматом попадает и на экран «Рост», и в еженедельный
  отчёт (12e-2) — новых мест вживления НЕТ.
- Данные уже в кэше — 0 новых запросов к Telemetr, 0 AI, миграций НЕТ.

## Тесты (~8)
- insights: виральный пост в текущем окне НЕ переворачивает тренд вверх; выброс в
  прошлом окне не занижает базу; окна <4 постов — тренд как раньше.
- subscriberAnomaly: органический ряд → пусто; скачок +50 за день → аномалия;
  дырка в ряду → не ложный скачок; пустой ряд → пусто.
- marketSection: строка предупреждения есть при аномалиях / нет без них; без `*`/`_`.
